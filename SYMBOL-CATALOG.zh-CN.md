# ChemSymbolSearch Symbol Catalog

This catalog lists the DWG symbols included in ChemSymbolSearch, grouped by category/device library.

- Category ID is the number of a group, such as valve/fitting/pump symbols.
- Item No. is the number inside that category.
- Global ID is a stable number across the whole catalog.
- The 1/2/3 numbers shown by HGSYMFIND inside AutoCAD are search-result numbers for the current search only.
- When inserting in AutoCAD, use the result number currently shown in AutoCAD.

Total DWG symbols: 393
